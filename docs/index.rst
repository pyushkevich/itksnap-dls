ITK-SNAP Deep Learning Segmentation Server documentation
========================================================
This Python package interfaces with `ITK-SNAP <http://itksnap.org/>`_ to allow fast deep learning segmentation. Currently, the package provides an interface between ITK-SNAP and `nnInteractive <https://github.com/MIC-DKFZ/nnInteractive>`_, a 3D promptable segmentation model by Isensee et al.
